# Local ML Data

Dataset files are local inputs and are intentionally excluded from Git.

- `raw/`: downloaded source CSVs, retained locally for reproducibility
- `processed/`: normalized `text,label` data generated by `scripts/prepare_dataset.py`

Selected sources:

- [SpaPhish v5](https://doi.org/10.17632/hz2d6gz7pc.5), CC BY 4.0, 1,395 Spanish real-world anonymized emails (664 legitimate, 731 phishing)
- [Phishing validation emails](https://doi.org/10.5281/zenodo.13474746), CC BY 4.0, 2,000 English real/synthetic rows (1,000 per class), only 100 unique texts after cleaning

Required processed contract:

- `text`
- `label`: `0 = legitimate`, `1 = phishing`

Preparation removes empty text and exact duplicate normalized text before the training split. Full emails, processed corpora, model artifacts, and generated reports must not be committed.
